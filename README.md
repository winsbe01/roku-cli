# roku-cli
a command line remote for Roku

## usage
- [h]ome
- [b]ack
- [p]lay
- [r]eplay
- * (star)
- arrow keys
- enter key
- < and > (rew and ff)
- / (switch to typing mode, 'enter' to exit)

## install
replace the default IP address in `roku.config` with the IP address of 
the roku. then run:

```
make
sudo make install
```

program can be started using `roku`
